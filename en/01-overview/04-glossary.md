# Glossary — Central Membership & SSO Hub

This document contains technical and business terms used in the Central Membership & SSO Hub system.

## A

### Member Account

The primary identity of a user within the SaaS ecosystem. One member account can have multiple licenses for different products.

### License Activation

The process of activating a user's access rights to a SaaS product. Activation can occur automatically (free plan) or after successful payment (paid plan).

### Audit Log

A record of every important action that occurs within the system, including who performed it, when, and what was done.

## C

### Callback / Webhook

Automatic notification sent by the payment gateway to the system after a transaction is completed (successful or failed). Callback is used to trigger license activation automatically.

### Checkout

The process of package selection and payment completion by the user.

## G

### Grace Period

A grace period after a paid subscription ends but the user has not yet renewed. During the Grace Period, users can still access the application with a warning, but after this period ends, access will be suspended.

## I

### Invoice

Proof of payment transaction sent to the user's email after successful payment.

### JWT (JSON Web Token)

An industry-standard security token issued by the Membership Hub after a user successfully logs in. This token contains encrypted information about the user's identity, licensed products, and validity period.

The JWT token is verified by each SaaS application to ensure the user has legitimate access.

## L

### License-ID

A unique code generated by the system for each active subscription belonging to a user for a specific product.

Format:

```
[PREFIX_PRODUK]-[XXXX]-[XXXX]-[XXXX]
```

Example:

```
NTO-A1B2-C3D4-E5F6
```

One user can have multiple License-IDs if they subscribe to several products.

### License

Official access rights granted to a user to use a SaaS product based on the selected plan.

## M

### Member

A user who has an account in the Membership Hub. A Member can subscribe to one or more SaaS products.

### Midtrans

An Indonesian payment gateway that supports various payment methods, including bank transfers, credit cards, and digital wallets.

## O

### OAuth2

An industry-standard authorization protocol used as the basis for SSO implementation. It allows third-party applications (SaaS app) to gain access on behalf of users without knowing their passwords.

## P

### Plan

A level of service offered for a SaaS product. Each product defines its own plan.

Examples of plan types:

- **Free**: Active without payment.
- **Pro**: Paid monthly or annually.
- **Business**: Paid with more complete features.

### Payment Gateway

A third-party service that processes payment transactions between users and the system.

### Product

Each SaaS application registered in the ecosystem and accessible via the Membership Hub.

### Product Code

A short code that identifies each SaaS product.

Example:

- `NTO` for NOTO.

This code is used as a prefix for the License-ID.

## R

### Renewal

The process of extending a paid subscription after its active period expires. Can be done before or during the Grace Period.

## S

### SSO (Single Sign-On)

A mechanism that allows users to log in once to the Membership Hub and gain access to all applications with an active license without having to log in separately.

### Subscription

An active subscription owned by a user for a product. Each subscription has a status, plan, active period, and License-ID.

### Suspend

The status of an account or license that is suspended because the paid subscription has ended and the Grace Period has expired. Users cannot access the application while in suspend status.

## T

### Tier / Plan Level

The level of a plan that determines the features and limitations available to users.

### Token

See **JWT**.

## V

### Email Verification

The process of confirming that the email used during registration is valid and owned by the user, via a link sent to that email.

## W

### Webhook

See **Callback**.

### Xendit

An Indonesian payment gateway that supports various payment methods, including bank transfers, e-wallets, credit cards, and QRIS.

## Terms Not To Be Confused

### License-ID and Password

- License-ID is a product license identifier code, not a password replacement.
- Password is the login credential for a member account.

### Plan and Product

- A Product is a licensed SaaS application.
- A Plan is a service level within one product.

### Suspend and Delete Account

- Suspend temporarily revokes access.
- Delete account permanently removes all data (if permitted).

### Grace Period and Trial

- Grace Period is a grace period after a paid subscription ends.
- Trial is a trial period before a user decides to subscribe.
