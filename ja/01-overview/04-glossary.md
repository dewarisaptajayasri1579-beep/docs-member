# 中央会員権 & SSO ヒュブ

このドキュメントには、システム Central Membership & SSO Hub で使用される技術的およびビジネス用語が記載されています。

## A

### 会員アカウント

SaaS エコシステムにおけるユーザーの主な識別情報。1 つの会員アカウントは、異なる製品に対して複数のライセンスを持つことができます。

### ライセンスアクティベーション

ユーザーが特定の SaaS製品に対するアクセス権を取得するプロセス。アクティベーションは、自動的に発生する無料パッケージ (free plan) または有料パッケージ (paid plan) の場合に発生します。

### オーディットログ

システム内で重要なアクションが発生したときに記録されるログ。ログには、ユーザー ID、日時、実行されたアクションが記載されています。

## C

### コールバック / ウェブフック

支払いゲートウェイがトランザクションが完了したときにシステムに自動的に送信する通知。コールバックは、ライセンスの自動アクティベーションをトリガーします。

### チェックアウト

ユーザーがパッケージを選択し、支払いを完了するプロセス。

## G

### グレースペリオッド

有料サブスクリプションが終了した後、ユーザーがアクセスを維持できる期間。グレースペリオッド中、ユーザーはアプリケーションにアクセスできますが、期間が終了するとアクセスが停止します。

## I

### インバイス

ユーザーに送信される、支払いが成功したときのトランザクションの証明書。

## J

### JWT (JSON ウェブ トークン)

会員権ヒュブがユーザーがログインに成功したときに発行する、標準的なセキュリティ トークン。トークンには、ユーザー ID、ライセンスされた製品、有効期間などの暗号化された情報が含まれます。

トークンは、各 SaaS アプリケーションによって検証され、ユーザーが有効なアクセス権を持っていることを確認します。

## L

### ライセンス ID

システムが生成する、ユーザーが有効なサブスクリプションを持つ製品ごとの一意のコード。

フォーマット:

```
[製品プレフィックス]-[XXXX]-[XXXX]-[XXXX]
```

例:

```
NTO-A1B2-C3D4-E5F6
```

ユーザーは、複数の製品に対して複数のライセンス ID を持つことができます。

### ライセンス

ユーザーが特定の SaaS製品に対して有効なアクセス権を取得する権利。

## M

### メンバー

会員権ヒュブにアカウントを持つユーザー。メンバーは、1 つまたは複数の SaaS製品に対してサブスクリプションを持つことができます。

### ミドトランス

インドネシアの支払いゲートウェイ。ミドトランスは、銀行振込、クレジットカード、デジタルウォレットなど、多種多様な支払い方法をサポートしています。

## O

### オーサー2

標準的な認証プロトコル。オーサー2は、第三者アプリケーション (SaaS アプリケーション) がユーザー名とパスワードを知らなくても、ユーザー名でアクセスできるようにします。

## P

### パッケージ (プラン)

SaaS製品に対して提供されるサービスレベル。各製品は独自のパッケージを定義します。

パッケージの例:

- **フリー / グラティス**: 無料で有効。
- **プロ**: 月額または年額の有料。
- **ビジネス**: 有料で、より多くの機能を提供します。

### 支払いゲートウェイ

ユーザーとシステムの間でトランザクションを処理する第三者サービス。

### プロダクト (製品)

SaaS製品が登録されているエコシステム内で、会員権ヒュブを介してアクセスできる製品。

### プロダクトコード

SaaS製品を一意に識別する短いコード。

例:

- `NTO` は NOTO です。

このコードは、ライセンス ID のプレフィックスとして使用されます。

## R

### リニューアル (パーペンジャン)

有料サブスクリプションが期限切れになったときに、ユーザーがサブスクリプションを更新するプロセス。リニューアルは、期限切れ前にまたはグレースペリオッド中に実行できます。

## S

### SSO (シングル サインオン)

ユーザーが会員権ヒュブでログインすると、すべてのライセンスが有効なアプリケーションにアクセスできるようにするメカニズム。

### サブスクリプション

ユーザーが持つ、特定の製品に対して有効なサブスクリプション。サブスクリプションには、ステータス、パッケージ、有効期間、ライセンス ID が含まれます。

### スサンド

サブスクリプションが期限切れになったときに、ユーザーがアクセスを維持できない状態。ユーザーはアプリケーションにアクセスできません。

## T

### タイヤ / レベル パッケージ

パッケージのレベルが、ユーザーに提供される機能と制限を決定します。

### トークン

JWT を参照してください。

## V

### バリファイ エメール

ユーザーが登録したメールアドレスが有効であることを確認するプロセス。確認のために、ユーザーにメールアドレスに送信されるリンクが含まれます。

## W

### ウェブフック

コールバックを参照してください。

### Xエンディット

インドネシアの支払いゲートウェイ。Xエンディットは、銀行振込、デジタルウォレット、クレジットカード、QRIS など、多種多様な支払い方法をサポートしています。

## 異なる用語を区別する

### ライセンス ID とパスワード

- ライセンス ID は、製品のライセンスを識別するコードであり、パスワードではありません。
- パスワードは、会員アカウントのログイン認証に使用されるセキュリティ情報です。

### パッケージと製品

- 製品は、SaaS製品を指します。
- パッケージは、製品ごとに提供されるサービスレベルです。

### スサンドとアカウントの削除

- スサンドは、アクセスを一時停止することです。
- アカウントの削除は、すべてのデータを永久に削除することです (許可されている場合)。

### グレースペリオッドとトライアル

- グレースペリオッドは、有料サブスクリプションが期限切れになった後、ユーザーがアクセスを維持できる期間です。
- トライアルは、ユーザーがサブスクリプションを購入する前に、製品を試用できる期間です。
